Pre-built functions provide a catalog of ready-to-use tasks or actions implemented using OCI functions.

1. Database Secret Rotation; with Wallet Pre-Built Function in OCI functions enhances your database security by automating secret rotation through empty LS connections. This function simplifies the process of maintaining secure database credentials.
2. document generator pre-built function; efficiently creates PDF documents from Office templates and JSON data. This function simplifies document creation by automating the process of combining template and data.
3. media workflow job spawner function; This function triggers media workflow jobs automatically when video content is uploaded to an object storage bucket. This function is particularly useful in scenarios such as encoding a video file into different resolutions immediately after it is uploaded into a bucket or encoding an audio file at various bitrates upon upload.
4. object storage file extractor pre-built function; This can be used to streamline your data management tasks. The function reads a zip file from an object storage bucket and extracts its content to a specified target bucket.
5. object storage file zip pre-built function; This function is designed to zip files stored in an OCI Object Storage bucket and save the resulting zip file to a specified target bucket
6. zero quota policy creator function; This function enforces spending limits by creating quota policies that prevent the creation of OCI resources in specified compartments. This function helps in managing OCI spending and ensuring budget adherence. Common use case scenarios here includes preventing resource creation after a certified spending threshold is met.
